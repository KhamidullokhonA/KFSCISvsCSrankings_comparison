KFSCIS Faculty → CSV

Extracts every person on cis.fiu.edu/faculty-staff — name, title, email, phone, profile link — into a spreadsheet-ready file.

Option A

One-click bookmarklet (recommended)

The directory is rendered by JavaScript, so the scraper has to run on the page itself. Drag this button to your bookmarks bar:

⬇ KFSCIS → CSV

Then open the faculty page, wait for it to load fully, and click the bookmark. A kfscis_faculty.csv download starts immediately.

Can't drag it? Copy the code into a bookmark manually

Right-click bookmarks bar → Add page → paste this as the URL.

Option B

Paste the page HTML here

On the loaded faculty page: open DevTools (F12) → Elements tab → right-click the top <html> node → Copy → Copy outerHTML → paste below. (Regular "View Source" won't work — it lacks the JS-rendered content.)
